Detailed Description

template<typename Tag, typename T>
struct DisAgg::Morpheme< Tag, T >

Zero-friction strong typedef over an underlying type T.

Template Parameters
TagAn incomplete struct used solely as a type discriminator. Two Morpheme instantiations with different Tag types are distinct types even if their T is identical.
TThe underlying value type. May be any copyable type.

Implicit conversion back to the underlying type.

Allows a Morpheme to be passed to any context that expects a T without an explicit cast. The compiler uses this conversion to resolve all operators, so no operator overloads are needed regardless of what T is.
