Detailed Description

Back-end contract for the weather domain.

Role: Back-end implementer
Write one retrieve and one save overload per weather type. No knowledge of structs, templates, or callers required. Passing a non-weather type to a weather front-end function is a compile error — this header being absent from scope is the entire enforcement mechanism.
